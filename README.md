# ML-DM-Stock-Price-Prediction

This repository contains the code and notebooks for stock price prediction using various machine learning and data mining techniques.

## Setup Instructions

To set up the environment and run the notebooks, follow these steps:

### 1. Create and Activate a Virtual Environment
Create a virtual environment to manage the dependencies for this project:

On Windows:

```bash
python -m venv env
.\env\Scripts\activate
```


### 2. Install Dependencies
Install the required packages using pip and the requirements.txt file:

```bash
pip install -r requirements.txt
```
